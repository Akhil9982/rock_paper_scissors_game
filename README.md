ğŸ’» Getting Started:

Follow these instructions to get a copy of the project up and running on your local machine.

Prerequisites:

You need a modern web browser (Chrome, Firefox, Safari, Edge) to play the game. No backend server or complex installation is required.

Installation:

Clone the repository:

Bash

git clone [https://github.com/Akhil9982/rock_paper_scissors_game.git](https://github.com/Akhil9982/rock_paper_scissors_game.git)

Navigate to the project directory:

Bash

cd rock_paper_scissors_game

Run the Game:

Simply double-click the index.html file to open it in your default browser.

OR, if you use VS Code, you can use the Live Server extension to launch it.

ğŸ•¹ï¸ How to Play

Open the game in your browser.

Click on one of the three images/buttons (Rock, Paper, or Scissors) to make your move.

The computer will simultaneously make its choice.

The result (Win, Lose, or Draw) will be displayed on the screen.

Check the scoreboard to see who is leading!
